get_j(jmax::FT, par::FT, curvature::FT, qy::FT)

Electron transport rate j, given

  • jmax Maximal eclectron transport @ leaf temperature (not 298.15 K)
  • par Absorbed PAR (photosynthetic active radiation)
  • curvature Curvature parameter (default at 0.9)
  • qy Quantum yield of electron, qy = maxPSII * PSII_frac, maxPSII is maximal PSII yield (default at 4.0/4.9), PSII_frac is Fraction of absorbed light used by PSII ETR (default at 0.5)

Land.PhotosynthesisModels.JmaxTDBernacchi — Type
JmaxTDBernacchi{FT} <:AbstractJmaxTD

A non-mutable AbstractJmaxTD type JmaxTDBernacchi that stores information for Jmax temperature correction. The equation used for temperature correction is Arrhenius equation (See Leuning (2002) "Leuning, R. "Temperature dependence of two parameters in a photosynthesis model"). Arrhenius correction over Jmax, given

  • para_set A JmaxTDBernacchi type parameter set
  • t_leaf Leaf temperature

The equations used are f_above = C * exp [ ( Ha /RT_0 ) * ( 1 - T_0/T_1 ) ], f_below = 1 + exp [ ( Sv*T_1 - Hd ) / ( RT_1) ], C = 1 + exp[(Sv*T_0 - Hd )/(RT_0 )], and correction = f_above /f_below
